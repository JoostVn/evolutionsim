## Description

This project simulates an evolutionary environment where bots (herbivores) use neural networks to navigate and survive. The neural networks are optimized using genetic algorithms (GA) over multiple generations. The simulation includes food supplies, barriers, and populations of bots that evolve to improve their fitness.

Key components:
- **Bots**: Herbivores with neural networks controlling their behavior.
- **Environment**: Contains food, barriers, and other objects.
- **Genetic Algorithm**: Optimizes the neural network weights through selection, crossover, and mutation.
- **Visualization**: Pygame-based GUI for real-time simulation viewing.
- **Analysis**: Tools for analyzing fitness evolution, gene similarity, and network visualization.
